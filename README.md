# stronghold-snake

The Snake sound pack for [peon-ping](https://github.com/PeonPing/peon-ping) — voice lines from The Snake in Stronghold Crusader.

## Install

```bash
peon packs install stronghold-snake
```

The `sounds/extras/` folder contains all remaining voice lines from The Snake that aren't mapped to a CESP category. They're included so anyone can remix or extend this pack without needing the original game files.
